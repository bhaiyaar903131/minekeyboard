This is my personal 81% mechanical keyboard 
using a Raspberry Pi Pico (RP2040) microcontroller, custom PCB, matrix wiring with diodes, USBC connectivity, and programmable firmware.

To use this , 
i will assemble all the parts and flash the included firmware onto the microcontroller. Once assembled and programmed, the keyboard functions as a fully customizable mechanical keyboard.


I built this keyboard to learn more about PCB design and hardware projects. 
It was also a fun way to create something useful for everyday typing and coding.
